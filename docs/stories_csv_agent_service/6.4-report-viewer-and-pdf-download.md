# Story 6.4: Report Viewer & PDF Download

## Status

Draft

## Story

**As a** user,
**I want** to view the generated NLP report and download it as PDF,
**so that** I can read and share analysis insights.

## Acceptance Criteria

1: Route: `/report/{id}` (protected, verify ownership)
2: `ReportViewer`: render Markdown content as styled HTML (react-markdown)
3: Language selector: toggle between Vietnamese and English (re-generate if needed)
4: Style selector: Summary vs Detailed
5: `PDFExport` button: download PDF, show loading spinner during generation
6: PDF preview: inline PDF viewer (iframe or react-pdf)
7: Share button: copy link to report
8: Print-friendly CSS for browser print

- `{frontend}/src/lib/api.ts` — API client

### Patterns & Conventions

- **Feature-based organization**: all report components in `components/report/`
- **Custom hooks**: `useReport` encapsulates report fetching, re-generation, PDF export
- **State management**: local state for language/style selection; API data via hook
- **Component naming**: PascalCase
- **Dynamic routing**: Next.js `[id]` parameter

### Tech Stack

- **Next.js 14** — App Router, dynamic route
- **React 18** — hooks
- **TypeScript 5.x** — type safety
- **react-markdown** — Markdown to HTML rendering with GFM support
- **react-pdf** (or iframe) — inline PDF viewing
- **Tailwind CSS** — typography, dark mode, print media queries
- **Axios** — API calls for report generation and PDF download

### Notes from Previous Stories

- Epic 4 (Story 4.1) provides `POST /api/v1/report/generate` — language and style params
- Epic 4 (Story 4.2) provides `POST /api/v1/report/{id}/export-pdf` and `GET /api/v1/report/{id}/download`
- Report content is Markdown stored in `reports.content` field
- PDF is stored in MinIO and streamed via download endpoint

### Testing

- Test location: `{frontend}/src/__tests__/` or colocated `*.spec.tsx`
- Framework: Jest + React Testing Library
- Mock API responses and `navigator.clipboard`
- Test types: unit tests for each component and hook behavior
