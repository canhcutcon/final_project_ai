# Story 1.1: Backend Project Scaffold

## Status

Draft

## Story

**As a** developer,
**I want** to scaffold the FastAPI backend project with proper folder structure,
**so that** the team has a standardized codebase to build upon.

## Acceptance Criteria

1. Cấu trúc `{backend}/app/` với các package: `api/v1/endpoints/`, `core/`, `models/`, `schemas/`, `services/`, `ml/models/`
2. `main.py` khởi động FastAPI app với CORS middleware, rate limiter
3. Endpoint `GET /health` trả về `{"status": "ok"}` với HTTP 200
4. `config.py` sử dụng Pydantic Settings đọc env vars: `DATABASE_URL`, `REDIS_URL`, `MINIO_ENDPOINT`, `JWT_SECRET`
5. `requirements.txt` bao gồm tất cả dependencies cần thiết (fastapi, uvicorn, sqlalchemy, celery, minio, pyjwt)
6. `.env.example` file mẫu cho tất cả env vars

### Patterns & Conventions

- **Architecture**: Layered — Controller (`endpoints/`) → Service (`services/`) → Repository (`models/`)
- **Dependency injection**: FastAPI `Depends()` via `api/deps.py`
- **All service methods**: `async def`
- **Naming**: `snake_case` cho files/functions/variables, `PascalCase` cho classes
- **Import order**: Standard library → Third-party → Local
- **API versioning**: Prefix `/api/v1/`
- **HTTP Status**: `200` success GET, `201` resource creation, `422` validation error

### Tech Stack

- **Python**: 3.11+
- **FastAPI**: latest — async, auto-docs (Swagger/ReDoc)
- **Pydantic Settings**: 2.x — env var management
- **SQLAlchemy**: 2.x — ORM (async support)
- **Uvicorn**: ASGI server
- **Database**: MySQL 8.0+ (via pymysql driver)
- **Linter**: Ruff

### Testing

- **Framework**: pytest
- **Test location**: `{backend}/tests/`
- **Test naming**: `test_*.py`
- **Minimum tests for this story**:
  - Test health endpoint returns 200 + correct body
  - Test config loads from env vars correctly
  - Test CORS middleware is configured
